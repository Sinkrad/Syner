The Synergy Project needs a good layout. (Topic Title)

This area handles discussion threads in a fashion very similar to a single forum. There are no categories or subforums from this point since the category is the problem itself. Discussion threads are listed in order of most recent post. A discussion thread contains a title, content, and replies, that is all. Here users can create a discussion thread or reply to one. A discussion thread can be linked to from within the problem description in the form of an discussion icon. A discussion item can contain the same kind of content that the description or solutions can (source, problem, or solution icons).

Discussion items are one of the very few things that do not have collaborative editting. A user may edit their own item or reply, but not that of others.
