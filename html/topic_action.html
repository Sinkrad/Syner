The Synergy Project needs a good layout. (Topic Title)
History Box

Here we can list and manage user actions, these includes edits to any part of the problem page(title, description, tags), discussion items, solutions, and maybe votes(more on this on the vote page). Each action can be reported (which sends a notification to a moderator or admin) , and any moderator or admin action is also recorded here.

This area will mostly like need to be much larger than this box that I have given it. At the same time though I do like the comparison of past history and future events being right next to eachother.

Event Box

Here users can see recently planned events, and can also create their own.
Timeline Box

This is the line graph that displays the voting trends, and maps events and possibly edits onto the graph. I'm sure there are a lot of things we can map on the timeline like # of solution submissions, views, subscription count.
